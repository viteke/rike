Rike - Task#

Purpose#

Manage Tasks, milestones and artifacts

Understanding Task#

With no task selected, the portlet is empty. New parts can be created by clicking on 'new'. A task can be selected by searching for it in the search box, or clicking on a task in other portlets, like Overview or My Tasks.

Creating an artifact#

The artifact must a name, shortname and should have an URL pointing to some more information about the artifact.

Creating a milestone#

The milestone must a name and should have an URL pointing to some more information about the artifact. Furthermore an enddate can be assinged and a release may be selected.

Working with tasks#

Creating a task#

The milestone must a name and should have an URL pointing to some more information about the artifact. An artifact must be selected the new task is assigned to. The size is the estimation of the LOC that need to be done. After creating a new task, the task must be evaluated.

Evaluating a task#

While evaluating you can correct the milestone of the task. Furthermore a priority and a complexity must be set. The size of the task may also be adjusted. After evaluating a task, it can either be started by someone else, or yourself after 24h. A task may be evaluated again if it is open.

Starting a task#

Start a task by selecting an open one, and click start.

Finishing a task#

By selecting a task, you are currently working on, you may close the task. You must enter the time (h) you put into the task and the LOC you produced while working on it.
